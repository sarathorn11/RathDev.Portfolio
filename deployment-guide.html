Deploy to GitHub Pages

A complete interactive guide to hosting your React/Vite portfolio on GitHub for free.

1 Project Setup
2 Install Tools
3 Configure
4 Git Push
5 Launch
01
Initial Project Setup

Ensure your project is ready for production. Since you are using Vite, we need to verify the build configuration first.

Verify Vite Config

In your vite.config.js, make sure you have the base property set to your repository name. This ensures assets are loaded correctly from the sub-path.

export default defineConfig({
  base: '/RathDev.Portfolio/',
  plugins: [react()],
  // ... rest of your config
})
💡
The base path must match your GitHub repository name exactly, including case-sensitivity.
02
Install gh-pages

We use the gh-pages package to automate the process of pushing our production build to a specific branch on GitHub.

Install Dependency

Run this command in your terminal at the root of your project:

npm install gh-pages --save-dev
⚠️
Make sure you are in the root directory of your project (where package.json is) before running the command.
03
Configure Scripts

Update your package.json to include the deployment URL and the necessary scripts to build and deploy.

Add Homepage & Scripts

{
  "name": "rathdev-portfolio",
  "homepage": "https://sarathorn11.github.io/RathDev.Portfolio/",
  "scripts": {
    "dev": "vite",
    "build": "vite build",
    "predeploy": "npm run build",
    "deploy": "gh-pages -d dist",
    "preview": "vite preview"
  }
}

The predeploy script runs automatically before deploy, ensuring your latest changes are always built.

04
Push to GitHub

Commit your changes and push them to your main branch on GitHub.

Git Commands

git add .
git commit -m "Setup deployment for GitHub Pages"
git push origin main
⚡
Your source code is now safely on GitHub. Now let's trigger the actual deployment.
05
Deploy & Enable

The final step! Run the deploy command and then verify the settings on GitHub.

Run Deploy

npm run deploy

GitHub Settings

Go to your repository on GitHub:

  1. Click Settings tab.
  2. Click Pages in the sidebar.
  3. Under "Build and deployment", ensure Source is Deploy from a branch.
  4. Branch should be set to gh-pages and folder to / (root).
🎉
Your site should be live at:
https://sarathorn11.github.io/RathDev.Portfolio/
